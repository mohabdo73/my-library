## üìñ About The Project
**My Digital Library** is a modern web application built with Django. It serves as an e-commerce platform for browsing and purchasing books. The project features a clean user interface with RTL (Right-to-Left) support for Arabic content and a dynamic shopping cart system.

## ‚ú® Key Features

- **Browse Books:** View a list of available books with cover images and prices.
- **Book Details:** Dedicated pages for each book with detailed descriptions.
- **Smart Shopping Cart:**
  - Add items to the cart without logging in (Session-based).
  - Dynamic total price calculation.
  - View and manage cart items.
- **Admin Panel:** Full control for administrators to add, edit, or delete books.
- **Responsive Design:** Fully responsive layout using Bootstrap 5.
